# Proyecto: Alka Wallet

## Descripción General
Alka Wallet es una billetera digital enfocada en el diseño y la estructura visual de la interfaz de usuario. En esta etapa, el objetivo principal es crear las pantallas siguiendo los requerimientos estéticos y de comportamiento visual, sin implementar la lógica de conexión o backend.

## Objetivos
- Desarrollar una interfaz de usuario atractiva y funcional.
- Definir la estructura visual de cada pantalla.
- Asegurar que el comportamiento visual de los componentes sea el adecuado.
