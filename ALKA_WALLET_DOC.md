# Proyecto: Alka Wallet

## Descripción General
Alka Wallet es una billetera digital enfocada en el diseño y la estructura visual de la interfaz de usuario. En esta etapa, el objetivo principal es crear las pantallas siguiendo los requerimientos estéticos y de comportamiento visual, sin implementar la lógica de conexión o backend.

## Objetivos
- Desarrollar una interfaz de usuario atractiva y funcional.
- Definir la estructura visual de cada pantalla.
- Asegurar que el comportamiento visual de los componentes sea el adecuado.

## Pantallas
1. **Splash Screen (Pantalla de inicio)**
   - **Función:** Aparece al abrir la app por un tiempo corto (2-3 segundos). Muestra el logo o bienvenida.
   - **Diseño:** Logo o nombre de la aplicación, con una animación o un fondo de color.
2. **Login / Signup Page**
   - Alternancia entre inicio de sesión y registro mediante Tabs.
   - Validación básica y simulación de flujo.
5. **Home Page**
   - Resumen de saldo, acciones rápidas y lista de movimientos.
6. **Home Page - Empty Case**
   - Variación de la Home Page para nuevos usuarios.
   - Mensajes de guía y botón para agregar la primera tarjeta.
7. **Profile Page (Pantalla de perfil)**
   - Visualización y edición de información personal (nombre, correo).
   - Opciones de seguridad, notificaciones y cierre de sesión.
8. **Send Money (Pantalla para enviar dinero)**
   - Formulario para ingresar monto y destinatario.
   - Diálogo de confirmación de éxito tras el envío.
